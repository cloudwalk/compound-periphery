This repository contains [Compound protocol](https://github.com/cloudwalk/compound-protocol) periphery smart contracts.

## Build and test

```sh
# Install all dependencies
npm install

# Compile all contracts
npx hardhat compile

# Run all tests
npx hardhat test
```
